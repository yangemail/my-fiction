欢迎使用JEECMS v9.3版，请按照以下步骤进行软件安装：

一、安装运行环境（JDK7+TOMCAT7+MYSQL5及以上，具体安装过程请参考本下载包中的相关说明文档）;

二、将程序解压后的ROOT文件夹拷贝到tomcat安装目录下的webapps文件夹下（例如：D:\Tomcat7\webapps\），启动tomcat，在地址栏上输入http://localhost:8080(端口和部署路径视安装设置而定),请根据安装向导填写安装信息，安装完成后重启tomcat;

三、在浏览器的地址栏中输入http://localhost:8080，若系统正常显示网站首页则表示本系统顺利安装完成。


附：
